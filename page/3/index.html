概要 潜在语义分析

  • 潜在语义分析LSA是一种无监督学习方法，主要用于文本的话题分析，其特点是通过矩阵分解发现文本域单词之间的基于话题的语义关系。
  • 文本信息处理中，传统的方法以单词向量表示文本的语义内容，以单词向量空间的度量表示文本之间的语义相似度。潜在语义分析旨在解决这种方法不能准确表示的语义的问题，试图从大量的文本数据中发现潜在的话题，以话题向量表示文本的语义内容，以话题向量空间的度量更准确地表示文本之间语义相似度。
  • 潜在语义分析使用的是非概率的话题分析模型。具体地，将文本集合表示为单词-文本矩阵，对单词-文本矩阵进行奇异值分解，从而得到话题向量空间，以及文本在话题空间的表示。
  • 非负矩阵分解是另一种矩阵的因子分解方法，其特点是分解的矩阵非负。

17.1 单词向量空间与话题向量空间

17.1.1 单词向量空间

  • 文本信息处理的一个核心问题是对文本的语义内容进行表示，并进行文本之间的语义相似度计算。最简单的方法是利用 向量空间模型VSM，也就是单词向量空间模型。其基本想法是，给定一个文本，用一个向量表示该文本的“语义”，向量的每一维对应一个单词，其数值为该单词在该文本中出现的频数或权值；基本假设是文本中所有单词的出现情况表示了文本的语义内容；文本集合中的每个文本都表示为一个向量，存在于一个向量空间；向量空间的度量表示文本之间的“语义相似度”。
  • 给定一个含有$n$个文本的集合$D={d_1,d_2,…,d_n}$，以及在所有文本中出现的$m$个单词的集合$W={w_1,w_2,…,w_n}$。将单词在文本中出现的数据用一个单词-文本矩阵表示，记作$X=\left[ \begin{array}{1}x_{11}&x_{12}& … & x_{1n} \ x_{21}&x_{22}& … &x_{2n} \ … & … & … & … \ x_{m1} & x_{m2} & … & x_{mn} \end{array}\right]$。这是一个$m\times n$的矩阵，元素$x_{ij}$表示单词$w_i$在文本$d_j$中出现的频数或权值。这是一个稀疏矩阵。
  • 权值通常用单词频率-逆文本频率表示，定义为$TFIDF_{ij}=\frac{tf_{ij}}{tf_{\cdot j}}log\frac{df}{df_i},i=1,2,…,m;j=1,2,…,n$式子中，$tf_{ij}$是单词$w_i$出现在文本$d_j$中的频数，$tf_{\cdot j}$是文本$d_j$中出现的所有单词的频数之和，$df_i$是含有单词$w_i$的文本数，$df$是文本集合$D$的全部文本数。直观上，一个单词在一个文本中出现的频数越高，这个单词在这个文本中的重要度就越高；一个单词在整个文本中出现的文本数越少，这个单词就越能代表其所在文本的特点，重要度就越高；一个单词在一个文本的TF-IDF是两种重要度的积，表示综合重要度。
  • 单词向量空间模型直接使用单词-文本矩阵的信息。单词-文本矩阵的第$j$列向量$x_j$表示文本$d_j$，$x_j=\left[\begin{array}{1} x_{1j} \ x_{2j} \ … \ x_{mj} \end{array} \right],j=1,2,…,m$，其中，$x_{ij}$是单词$w_i$在文本$d_j$的权值，$i=1,2,…,m$权值越大，该单词在该文本中的重要度就越高。矩阵$X=[x_1 \space x_2 \space …\space x_n]$。
  • 两个单词向量的内积或标准化内积表示对应的文本之间的语义相似度，文本$d_i$与$d_j$之间的相似度为$x_i\cdot x_j,\frac{x_i\cdot x_j}{||x_i||||x_j||}$，式子中，$\cdot$表示向量的内积，带上双竖线表示其范数。

17.1.2 话题向量空间

话题向量空间

给定一个文本集合$D={d_1,d_2,…,d_n}$和一个相应的单词集合$W={w_1,w_2,…,w_m}$可以获得其单词-文本矩阵$X$，$X$构成原始的单词向量空间，每一列都是一个文本在单词向量空间的表示。

假设所有文本共包含$k$个话题，假设每个话题由一个定义在单词集合$W$上的$m$维向量表示，称为话题向量，即$t_l=\left[\begin{array}{1}t_{1l} \ t_{2l} \ … \ t_{ml} \end{array}\right],l=1,2,…,k$。其中，$t_{il}$是单词$w_i$在话题$t_l$的权值，权值越大，该单词在该话题下重要度就越高。这$k$个话题向量张成一个话题空间向量，维数为$k$。向量空间$T$也可以表示成一个矩阵，称为单词-话题矩阵。

文本在话题向量空间的表示

考虑文本集合$D$的文本$d_j$，在单词向量空间中由一个向量$x_j$表示，将$x_j$投影到话题向量空间$T$中，得到在话题空间的一个向量$y_j$，$y_j$是一个$k$维向量，其表达式为$y_j=\left[\begin{array}{1}y_{1j} \ y_{2j} \ … \ y_{kj}\end{array}\right],j=1,2,…,n$其中$y_{lj}$是文本$d_j$在话题$t_l$的权值，权值越大代表该话题在该文本中的重要度越大。

矩阵$Y$表示话题在文本中的情况，称为话题-文本矩阵。

单词向量空间到话题向量空间的线性变换

在单词向量空间的文本向量$x_j$可以通过它在话题空间中的向量$y_j$近似表示，具体地由$k$个话题向量以$y_j$为系数的线性组合近似表示$x_j\approx y_{1j}t_1+y_{2j}t_2+…+y_{kj}t_k,j=1,2,…,n$，所以，单词-文本矩阵$X$可以近似的表示为单词话题矩阵$T$与话题文本矩阵$Y$的乘积$X\approx TY$。这就是潜在语义分析。

直观上潜在语义分析是将文本在单词向量空间的表示通过线性变换转换为在话题向量空间中的表示。

17.2 潜在语义分析算法

单词—文本矩阵

给定文本集合$D$和单词集合$W$。潜在语义分析首先将这些数据表示成一个单词—文本矩阵。其为一个$m\times n$矩阵，元素$x_{ij}$表示单词$w_i$在文本$d_j$中的出现频数或权值。

截断奇异值分解

潜在语义分析根据确定的话题个数$k$对单词—文本矩阵$X$进行奇异值分解，$X\approx U_k{\sum}_kV_k^T=[u_1 \space \space u_2 \space \space … \space \space u_k]\left[\begin{array}{1}\sigma_1&0&0&0 \ 0&\sigma_2&0&0 \ 0&0& … & 0 \ 0 & 0&0& \sigma_k\end{array} \right]\left[\begin{array}{1}v_1^t \ v_2^T \ … \ v_k^T\end{array} \right]$，式中$k\leq n\leq m$，$U_k$是$m\times k$矩阵，它的列由$X$得到前$k$个互相正交的左奇异向量组成，${\sum}_k$是$k$阶对角方阵，对角元素为前$k$个最大的奇异值，$V_k$是$n\times k$矩阵，它的列由$X$的前$k$个互相正交的右奇异向量组成。

话题向量空间

在单词—文本矩阵$X$的截断奇异值分解式中，矩阵$U_k$的每一个列向量$u_i$都表示一个话题，称为话题向量，这$k$个向量张成一个子空间$U_k=[u_1 \space \space u_2 \space \space … \space \space u_k]$称为话题向量空间。

17.3 非负矩阵分解算法

17.3.1 非负矩阵分解

若一个矩阵的所有元素非负，则称该矩阵为非负矩阵，若$X$是非负矩阵，记作$X\geq 0$。给定一个非负矩阵$X\geq0$，找到两个非负矩阵$W\geq0$和$H\geq0$使得$X\approx WH$。即将非负矩阵$X$分解为两个非负矩阵$W$和$H$的乘积的形式，称为非负矩阵分解。很难完全相等，所以只能近似相等。

假设非负矩阵$X$是$m\times n$矩阵，非负矩阵$W$和$H$分别为$m\times k$矩阵和$k\times n$矩阵。假设$k<min(m,n)$，即$W$和$H$小于原矩阵$X$，所以非负矩阵分解是对原数据的亚索。矩阵$X$的第$j$列向量$x_j$满足$x_j\approx Wh_j=[w_1\space \space w_2 \space \space … \space \space w_k]\left[\begin{array}{1}h_{1j} \ h_{2j} \ … \ h_{kj} \end{array}\right]=\sum_{l=1}^kh_{lj}w_l,j=1,2,…,n$，其中，$h_j$是矩阵$H$的第$j$列，$w_l$是矩阵$W$的第$l$列，$h_{lj}$是$h_j$的第$l$个元素。矩阵$X$的第$j$列$x_j$可以由矩阵$W$的$k$个列$w_l$的线性组合逼近，线性组合的系数是矩阵$H$的第$j$列$h_j$的元素。矩阵$W$的列向量是一组基，矩阵$H$的列向量为线性组合系数。称$W$为基矩阵，$H$为系数矩阵。非负矩阵分解旨在用较少的基向量、系数向量来表示较大的数据矩阵。

17.3.2 潜在语义分析

给定一个$m\times n$非负的单词—文本矩阵$X\geq 0$。假设文本集合共包含$k$个话题，对$X$进行非负矩阵分解。即非负的$m\times k$矩阵$W\geq 0$和$k\times n$矩阵$H\geq 0$，使得$X\approx WH$。令$W$为话题空间向量，表示文本集合的$k$个话题。$H$为文本在话题向量空间的表示。

17.3.3 非负矩阵分解的形式化

定义损失函数

平方损失，设两个非负矩阵$A=[a_{ij}]{m\times n}$和$B=[b{ij}]{m\times n}$，平方损失函数定义为$||A-B||^2=\sum{i,j}(a_{ij}-b_{ij})^2$，其下界是0，当且仅当$A=B$时达到下界。

散度，设两个非负矩阵$A=[a_{ij}]{m\times n}$和$B=[b{ij}]{m\times n}$，散度损失函数定义为$D(A||B)=\sum{i,j}(a_{ij}log\frac{a_{ij}}{b_{ij}}-a_{ij}+b_{ij})$，其下界是0，当且仅当$A=B$时到达下界。$A$和$B$不对称。当$\sum_{i,j}a_{ij}=\sum_{i,j}b_{ij}=1$时散度损失函数退化为Kullback-Leiber散度或相对熵。

目标函数$||X-WH||^2$关于$W$和$H$的最小化，满足约束条件$W,H\geq0$即$min_{W,H}||X-WH||^2,s.t.W,H\geq0$。或者，目标函数$D(X||WH)$关于$W$和$H$的最小化，满足约束条件$W,H\geq0$即$min_{W,H}D(X||WH),s.t. W,H\geq0$

17.3.4 算法

  • 平方损失$||X-WH||^2$对下列乘法更新规则$H_{lj}\leftarrow H_{lj}\frac{(W^TX){lj}}{(W^TWH){lj}},W_{il}\leftarrow W_{il}\frac{(XH^T){il}}{(WHH^T){il}}$，是非增的。当且仅当$W$和$H$是平方损失函数的稳定点时函数的更新不变。

  • 散度损失$D(X-WH)$对下列乘法更新规则$H_{lj}\leftarrow H_{lj}\frac{\sum_i[w_{il}X_{ij}/(WH){ij}]}{\sum{i}W_{il}},W_{il}\leftarrow W_{il}\frac{\sum_j[H_{lj}X_{ij}/(WH){ij}]}{\sum_jH{lj}}$是非增的。当且仅当$W$和$H$是散度损失函数的稳定点时函数的更新不变。

读书笔记-统计学习方法第十六章

概要 主成分分析

  • 主成分分析是一种常用的无监督学习方法，这一方法利用正交变换把由线性相关变量表示的观测数据转换为少数几个由线性无关变量表示的数据，线性无关的变量称为主成分。主成分的个数通常小于原始变量的个数，所以主成分分析属于降维方法。
  • 主成分分析主要用于发现数据中的基本结构，即数据中变量之间的关系，是数据分析的有力工具，也用于其他机器学习方法的前处理。

16.1 总体主成分分析

16.1.1 基本想法

对给定数据进行规范化，使得数据每一变量的平均值为0，方差为1。对数据进行正交变换，原来由线性相关变量表示的数据，通过正交变换变成由若干个线性无关的新变量表示的数据。新变量是可能的正交变换中变量的方差的和最大的，方差表示在新变量上信息的大小。将新变量一次成为第一主成分，第二主成分等。

16.1.2 定义和导出

假设$x = (x_1,x_2,…,x_m)^T$是$m$维随机变量，其均值向量是$\mu=E(x)=(\mu_1,\mu_2,…,\mu_m)^T$。协方差矩阵$\sum=cov(x,x)=E[(x-\mu)(x-\mu)^T]$。考虑由$m$维随机变量$x$到$m$维随机变量$y=(y_1,y_2,…,y_m)^T$的线性变换$y_i=\alpha_i^Tx=\alpha_{1i}x_1+\alpha_{2i}x_2+….+\alpha_{mi}x_m$。其中$\alpha_i^T=(\alpha_{1i},\alpha_{2i},…,\alpha_{mi}),i=1,2,…,m$。由随机变量性质可以知道$E(y_i)=\alpha_i^T\mu,i=1,2,…,m$，$var(y_i)=\alpha_i^T\sum{\alpha_i},i=1,2,…,m$，$cov(y_i,y_j)=\alpha_i^T\sum{\alpha_j},i=1,2,…,m;j=1,2,…,m$

给出一个$y_i=\alpha_i^Tx=\alpha_{1i}x_1+\alpha_{2i}x_2+….+\alpha_{mi}x_m$线性变换，如果满足条件（1）系数向量$\alpha_i^T$是单位向量，即$\alpha_i^T\alpha_i=1,i=1,2,…,m$（2）变量$y_i$与$y_j$互不相关，即$cov(y_i,y_j)=0(i\neq j)$（3）变量$y_1$是$x$的所有线性变换中方差最大的；$y_2$是与$y_1$不相关的$x$的所有线性变换中方差最大的；一般地，$y_i$是与$y_1,y_2,…,y_{i-1}$都不相关的$x$的所有线性变换中方差最大的；此时，分别称$y_1$是$x$的第一主成分，$y_2$是$x$的第二主成分。。

16.1.3 主要性质

  • 设$x$是$m$维随机变量，$\sum$是$x$的协方差矩阵，$\sum$的特征值分别是$\lambda_1\geq \lambda_2\geq…\geq\lambda_m\geq 0$，特征值对应的单位特征向量分别是$\alpha_1,\alpha_2,…,\alpha_m$，则$x$的第$k$主成分是$y_k=\alpha_k^Tx=\alpha_{1k}x_1+\alpha_{2k}x_2+…+\alpha_{mk}x_m,k=1,2,…,m$，$x$的第$k$主成分的方差是$var(y_k)=\alpha_k^T\sum{\alpha_k}=\lambda_k,k=1,2,..,m$，即协方差矩阵$\sum$的第$k$个特征值。
  • $m$维随机变量$y=(y_1,y_2,..,y_m)^T$的分量依次是$x$的第一主成分到第$m$主成分的充要条件是（1）$y=A^Tx,A$为正交矩阵$A=\left[\begin{array}{1}\alpha_{11}&\alpha_{12}&…&\alpha_{1m}\ \alpha_{21}&\alpha_{22}&…&\alpha_{2m}\ … &…&…&…\ \alpha_{m1}&\alpha_{m2}&…&\alpha_{mm} \end{array} \right]$（2）$y$的协方差矩阵为对角矩阵$cov(y)=diag(\lambda_1,\lambda_2,…,\lambda_m),\lambda_1\geq\lambda_2\geq…\geq\lambda_m$，其中$\lambda_k$是$\sum$的第$k$个特征值，$\alpha_k$是对应的单位特征向量，$k=1,2,…,m$
  • 总体主成分的性质：（1）总体主成分$y$的协方差矩阵是对角矩阵$cov(y)=\wedge=diag(\lambda_1,\lambda_2,…,\lambda_m)$（2）总体主成分$y$的方差之和等于随机变量$x$的方差之和，即$\sum_{i=1}^m\lambda_i=\sum_{i=1}^m\sigma_{ii}$其中$\sigma_{ii}$是随机变量$x_i$的方差，即协方差矩阵$\sum$的对角元素。可知$\sum_{i=1}^mvar(x_i)=tr(\sum^T)=tr(A\wedge A^T)=tr(A^T\wedge A)=tr(\wedge)=\sum_{i=1}^m\lambda_i=\sum_{i=1}^mvar(y_i)$（3）第$k$个主成分$y_k$与变量$x_i$的相关系数$\rho(y_k,x_i)$称为因子负荷量，它表示第$k$个主成分$y_k$与变量$x_i$的相关关系，计算公式为$\rho(y_k,x_i)=\frac{\sqrt \lambda_k\alpha_{ik}}{\sqrt \sigma_{ii}},k,i=1,2,…,m$（4）第$k$个主成分$y_k$与$m$个变量的因子负荷量满足$\sum_{i=1}^m\sigma_{ii}\rho^2(y_k,x_i)=\lambda_k$（5）$m$个主成分与第$i$个变量$x_i$的因子负荷量满足$\sum_{k=1}^m\rho^2(y_k,x_i)=1$

16.1.4 主成分的个数

  • 对任意正整数$q$，$1\leq q\leq m$，考虑正交线性变换$y=B^Tx$其中$y$是$q$维向量，$B^T$是$q\times m$矩阵，令$y$的协方差矩阵为$\sum_y=B^T\sum B$，则$\sum_y$的迹$tr(\sum_y)$在$B=A_q$的时候取最大值，其中矩阵$A_q$由正交矩阵$A$的前$q$列组成。
  • 考虑正交变换$y=B^Tx$，这里$B^T$是$p\times m$矩阵，$A$和$\sum_y$的定义与上一个相同，则$tr(\sum_y)$在$B=A_p$时取得最小值，其中矩阵$A_p$由$A$的后$p$列组成。
  • 第$k$主成分$y_k$的方差贡献率定义为第$k$个方差之和与所有方差之和的比$\eta_k=\frac{\lambda_k}{\sum_{i=1}^m\lambda_i}$，$k$个主成分$y_1,y_2,…,y_k$的累计方差贡献率定义为$k$个方差之和与所有方差之和的比$\sum_{i=1}^k\eta_i=\frac{\sum_{i=1}^k\lambda_i}{\sum_{i=1}^m\lambda_i}$。
  • $k$个主成分$y_1,y_2,…,y_k$对原有变量$x_i$的贡献率定义为$x_i$与$(y_1,y_2,…,y_k)$的相关系数的平方，记作$v_i=\rho^2(x_i,(y_1,y_2,…,y_k))=\sum_{j=1}^k\rho^2(x_i,y_j)=\sum_{j=1}^k\frac{\lambda_j\alpha_{ij}^2}{\sigma_{ii}}$

算法思想

Q-Learning是强化学习中value-based的算法，$Q$即为$Q(s,a)$，就是在某一个时刻状态state，采取动作action就能获得的收益的期望，环境会根据agent动作反馈相应的reward。算法就是根据state和action构建一张Q_Table来存储所有状态和动作的期望收益表，根据这个表选取在某一状态下能获得的最大收益的动作。

缺点与改进

大部分情况下，问题的状态空间和动作空间都很巨大，如果建立一个Q表存在内存之中从时间和空间两个角度来看都是不行的，所以提出了Deep Q-Learning算安抚。

Deep Q-Learning算法

算法原理

$Q(s,a)$是程序的记忆，通过记忆，计算在状态$s$下，执行操作$a$的权值。每次在操作集合$A$中选择使$Q$最大的$a$操作执行，即$argmax_{a\in A}Q(s,a)$。

$Q$的转移公式为$Q(s,a)=R(s,a)+\gamma max_{\widetilde a}Q(\widetilde s,\widetilde a)$。$R(s,a)$为在状态$s$下，执行操作$a$获得的奖励；$\widetilde s$为状态$s$执行操作$a$后的新状态；$Q(\widetilde s,\widetilde a)$为新状态的$Q$值；$\gamma$为折扣因子，可以看出$\gamma$越大，程序越看重记忆中的经验价值，反之，则越在乎眼前的利益；对于普通的Q-learning算法Q函数用一张表格表示，在DQN中，Q函数为神经网络。

值函数近似

该方法是为了解决状态空间过大，也是维度灾难的问题。通过用函数而不是Q表来表示$Q(s,a)=f(s,a)$，这个函数可以是线性的也可以是非线性的。比如$Q(s,a)=w_1s+w_2a+w_3$，把其中的$w$全部提出来当做参数，就可以写成$f(s,a,w)$。

其中，$w$是权重，对输入的状态提取特征作为输入，然后用神经网络去训练他。

使用方差损失函数作为损失函数$J(w)=E_\pi[(r+\gamma maxQ(s’,a’,w)-Q(s,a,w))^2]$​，使用随机梯度下降法来迭代求解。其中，真实的Q值大概是$R+\lambda max_aQ(S_{t+1},a)$

算法流程

初始时，Q函数初始随机

1 随机选择一个状态$s$；2 按照一定探索概率$\epsilon$随机选择可行操作，或者根据Q经验函数选择操作；3 用选择的操作$a$，得到下一个状态$\widetilde s$；4 更新$Q(s,a)$，对于普通Q-learning更新Q表格即可，对于DQN则从本次操作的$(s,a,\widetilde s)$存进经验库，当经验库的量足够大的时候从中选出若干条经验作为一个batch，训练一次神经网络。

对于探索率$\epsilon$，可以设置为1，之后随着迭代次数降低

算法原理

对于每一个状态，随机加上一点，然后形成新状态，检查是否更优，最后既可以找到局部最优解

爬山算法的基本思路是每次迭代时给当前取得的最优权重加上一组随机值，如果加上这组值使得有效控制倒立摆的持续时间变长了那么就更新它为最优权重，如果没有得到改善就保持原来的值不变，直到迭代结束。在迭代过程中，模型的参数不断得到优化，最终得到一组最优的权值作为控制模型的解。

遗传算法

概要

  • 遗传算法也称为进化算法，是受达尔文进化论的启发，借鉴生物进化过程而提出的一种启发式搜索。
  • 其主要特点是直接对结构对象进行操作，因此不同于其他求解最优解的算法，遗传算法不存在求导和对函数连续性的限定，采用概率化的寻优方法，不需要确定的规则就能自动获取和指导优化的搜索空间，自适应地调整搜索方向。

相关生物术语

  • 基因型：性状染色体的内部表现

  • 表现型：染色体决定的性状的外部表现，或者说，根据基因型形成的个体的外部表现

  • 进化：种群逐渐适应生存环境，品质不断得到改良。生物的进化是以种群的形式进行的

  • 种群：不同生物个体形成的群体，生物的进化以群体的形式进行，这样的一个群体称为种群

  • 个体：组成种群的单个生物
  • 基因：带有遗传信息的DNA片段，可以通俗的将基因理解为一段信息，这段信息决定的生物个体的性状
  • 适应度：生物个体对于生存环境的适应程度，越适应那么其得以存活和繁衍的概率就越大
  • 遗传：通过繁殖过程，子代将从父母双方各获取一部分基因，形成新的自己的基因，这个过程中，会发生基因的复制、交叉，也会以较低的概率发生基因突变
  • 自然选择：物竞天择，适者生存的自然淘汰机制。具体为对环境适应度高的个体参与繁殖的机会比较多，后代就会越来越多。适应度低的个体参与繁殖的机会比较少，后代就会越来越少
  • 复制：细胞分裂时，遗传物质DNA通过复制而转移到新产生的细胞中，新细胞就继承了旧细胞的基因
  • 交叉：两个染色体的某一相同位置处DNA被切断，前后两串分别交叉组合形成两个新的染色体。也称基因重组或杂交
  • 变异：复制时可能（很小的概率）产生某些复制差错，变异产生新的染色体，表现出新的性状
  • 编码：DNA中遗传信息在一个长链上按一定的模式排列。遗传编码可看作从表现型到基因型的映射
  • 解码：基因型到表现型的映射

通俗解释

有很多袋鼠，它们降落到喜玛拉雅山脉的任意地方。这些袋鼠并不知道它们的任务是寻找珠穆朗玛峰。但每过几年，就在一些海拔高度较低的地方射杀一些袋鼠。于是，不断有袋鼠死于海拔较低的地方，而越是在海拔高的袋鼠越是能活得更久，也越有机会生儿育女。就这样经过许多年，这些袋鼠们竟然都不自觉地聚拢到了一个个的山峰上，可是在所有的袋鼠中，只有聚拢到珠穆朗玛峰的袋鼠被带回了美丽的澳洲。

基本框架

编码

  • 由于遗传算法不能直接处理问题空间的参数,因此必须通过编码将要求解的问题表示成遗传空间的染色体或者个体。

    完备性：问题空间中的所有点(候选解)都能作为GA空间中的点(染色体)表现；健全性：GA空间中的染色体能对应所有问题空间中的候选解；非冗余性：染色体和候选解一一对应

  • 二进制编码

    就像人类的基因有AGCT 4种碱基序列一样。不过在这里我们只用了0和1两种碱基,然后将他们串成一条链形成染色体。一个位能表示出2种状态的信息量，因此足够长的二进制染色体便能表示所有的特征。这便是二进制编码。

  • 浮点编码法

    指个体的每个基因值用某一范围内的一个浮点数来表示。在浮点数编码方法中，必须保证基因值在给定的区间限制范围内，遗传算法中所使用的交叉、变异等遗传算子也必须保证其运算结果所产生的新个体的基因值也在这个区间限制范围内。

  • 符号编码法

    符号编码法是指个体染色体编码串中的基因值取自一个无数值含义、而只有代码含义的符号集。

适应度函数

  • 进化论中的适应度，是表示某一个体对环境的适应能力，也表示该个体繁殖后代的能力。遗传算法的适应度函数也叫评价函数，是用来判断群体中的个体的优劣程度的指标，它是根据所求问题的目标函数来进行评估的。
  • 遗传算法在搜索进化过程中一般不需要其他外部信息，仅用评估函数来评估个体或解的优劣，并作为以后遗传操作的依据。由于遗传算法中，适应度函数要比较排序并在此基础上计算选择概率，所以适应度函数的值要取正值。由此可见，在不少场合，将目标函数映射成求最大值形式且函数值非负的适应度函数是必要的。

初始群体选择

遗传算法中初始群体中的个体是随机产生的。一般来讲，初始群体的设定可采取如下的策略：根据问题固有知识，设法把握最优解所占空间在整个问题空间中的分布范围，然后，在此分布范围内设定初始群体；先随机生成一定数目的个体，然后从中挑出最好的个体加到初始群体中。这种过程不断迭代，直到初始群体中个体数达到了预先确定的规模。

运算过程

（1）初始化：设置进化代数计数器t=0，设置最大进化代数T，随机生成M个个体作为初始群体P(0)。

（2）个体评价：计算群体P(t)中各个个体的适应度。

（3）选择运算：将选择算子作用于群体。选择的目的是把优化的个体直接遗传到下一代或通过配对交叉产生新的个体再遗传到下一代。选择操作是建立在群体中个体的适应度评估基础上的。

（4）交叉运算：将交叉算子作用于群体。遗传算法中起核心作用的就是交叉算子。

（5）变异运算：将变异算子作用于群体。即是对群体中的个体串的某些基因座上的基因值作变动。群体P(t)经过选择、交叉、变异运算之后得到下一代群体P(t+1)。

（6）终止条件判断：若t=T,则以进化过程中所得到的具有最大适应度个体作为最优解输出，终止计算。

选择：从群体中选择优胜的个体，淘汰劣质个体的操作叫选择。选择算子有时又称为再生算子(reproduction operator)。选择的目的是把优化的个体(或解)直接遗传到下一代或通过配对交叉产生新的个体再遗传到下一代。选择操作是建立在群体中个体的适应度评估基础上的，常用的选择算子有以下几种：适应度比例方法、随机遍历抽样法、局部选择法。

交叉：在自然界生物进化过程中起核心作用的是生物遗传基因的重组(加上变异)。同样，遗传算法中起核心作用的是遗传操作的交叉算子。所谓交叉是指把两个父代个体的部分结构加以替换重组而生成新个体的操作。通过交叉，遗传算法的搜索能力得以飞跃提高。

变异：变异算子的基本内容是对群体中的个体串的某些基因座上的基因值作变动。依据个体编码表示方法的不同，可以有以下的算法：a)实值变异。b)二进制变异。

改进遗传算法

  • 染色体生成

    中间连接点的数量数量，不包括起点和目标点，独立于管道的要求和布线空间的特征。在布置单个管道的时候，中间连接点是为了提高初始路径的多样性和质量。当管道通过复杂的布管空间时，其值可能会略微增加。一般来说，从1到5的范围就足够了。当多个管道并行布线时，中间连接点可以改善管道之间的并行效果。如果平行路径的长度较长，可以适当增加其值，但不应该过大。

    在指定的起始点和目标点之间随机生成中间连接点。注意，连接点不能位于任何障碍物中，以确保管道路径的验证，指定的产生连接点的区域通常为起点和目标点为对角线的长方体。如果长方体空间中障碍物较少或空间过大，连接点的分布就会比较分散。这不利于在短时间内产生高质量的染色体。为了在实践中解决这种情况，需要进一步缩小连接点的产生空间。

    对中间连接点按其坐标的指定方向排序。

    使用A*算法在每个相邻连接点之间创建子路径，从而形成整个路由路径。

  • 选择算子和适应度函数

    选择算子根据指定的策略和每个染色体的适应度值从亲本染色体中选择后代染色体。它体现了遗传算法中的最优生存理论。采用轮盘赌方法作为选择策略，适用于求解最大优化问题。

    将目标函数$Min.f_{Objective}(path)=\alpha \times f_L(path) + \beta \times f_B(path)+\gamma \times f_E(path) - \delta \times f_N(path)$转换为$Max.f_{GA-Objective}(path)=K-f_{Objective}(path)$，其中$K$是一个较大的常数，保证其是一个正数。

    为了提高遗传算法的收敛性，在选择算子中采用精英主义，避免失去全局最优解。

  • 交叉算子

    交叉算子是遗传算法的主要进化动力。其通过母体染色体的基因产生后代染色体。交叉点是从母体染色体中随机选择的。其次，使用A*算法连接两个交叉点的子路径。最后，子路径和路径段的亲本染色体适当连接，形成后代染色体。

    该方法的染色体采用变长编码，无需额外转换即可直接映射到管道路径上。然而，由于交叉算子产生的子路径的不确定性，子代染色体的长度可能比亲本染色体长得多。如果没有对其进行约束处理，遗传算法种群的质量将逐步恶化。因此,本文方法提出了可接受的因素,也就是说,如果新的染色体的长度不大于可接受的产品因素和管道喷嘴的曼哈顿距离,新的染色体将投入后代一代,否则原父染色体将保持。可接受因子由用户设定，推荐值在[1.2,1.8]范围内。该方法能在一定程度上接受较差的解，可以用来平衡算法的优化能力和收敛能力。

  • 变异算子

    突变算子以较小的概率对染色体基因进行修饰，在不破坏进化收敛性的前提下保持种群的多样性。

    这有助于提高遗传算法的搜索性能。在染色体(P)中随机选择两个突变点，利用算法重新生成这两个突变点之间的子路径。最后，带有新的子路径的染色体进入后代染色体。同样，突变算子可以增加后代染色体的长度，因此这里也采用了交叉算子中的可接受因子法。

  • 修复操作符

    在染色体产生、交叉和突变的过程中，存在着获得重叠路径或循环路径染色体的可能性。有重叠或环路的管道路径虽然保持了起点和目标点之间的连续性，但处于非法状态，需要在演化过程中及时进行修复或丢弃。修复算子由两部分组成：去除重叠路径和去除环路路径。

    • 重叠路径去除

      假设路径网格信息，存储在数组路径中。该算法从起始网格开始依次遍历路径网格。如果发现路径$i-1$和路径$i+1$坐标在遍历过程中相等，就代表他俩重叠。就依次比较$i-2,i+2$，直到出现不等为止。调整调整，删除路径重叠部分。

概要 奇异值分解

  • 奇异值分解是一种矩阵因子分解方法，是线性代数的概念，主成分分析、潜在语义分析都用到了
  • 任意一个$m \times n$矩阵，都可以表示为三个矩阵的乘积（因子分解）分别是$m$阶正交矩阵、由降序排列的非负的对角线元素组成的$m\times n$矩形对角矩阵和$n$阶正交矩阵，称为该矩阵的奇异值分解。矩阵的奇异值分解一定存在，但不唯一。奇异值分解可以看做是矩阵数据压缩的一种方法，用因子分解的方式近似地表示原始矩阵，这种近似是在平方损失意义下的最优近似。

15.1 奇异值分解的定义与性质

15.1.1 定义与定理

  • 矩阵的奇异值分解是指，将一个非零的$m\times n$实矩阵$A$，$A\in R ^{m \times n}$，表示为以下三个实矩阵乘积形式的运算，即进行矩阵的因子分解：$A=U\sum V^T$。其中，$U$是$m$阶正交矩阵，$V$是$n$阶正交矩阵，$\sum$是由降序排列的非负的对角线元素组成的$m\times n$矩形对角矩阵吗，满足
    $$
    UU^T=I\
    VV^T=I\
    \sum = diag(\sigma_1,\sigma_2,…,\sigma_p)\
    \sigma_1 \geq\sigma_2\geq…\geq0 \
    p=min(m,n)
    $$
    其中，$U\sum V^T$称为矩阵$A$的奇异值分解，$\sigma_i$称为矩阵$A$的奇异值，$U$的列向量称为左奇异向量，$V$的列向量称为右奇异向量。

  • 若$A$为一$m\times n$实矩阵，$A\in R^{m\times n}$，则$A$的奇异值分解存在$A=U\sum V^T$。其中$U$是$m$阶正交矩阵，$V$是$n$阶正交矩阵，$\sum$是$m\times n$矩形对角矩阵，其对角线元素非负，且按降序排序。

15.1.2 紧奇异值分解与截断奇异值分解

  • 设有$m\times n$实矩阵$A$，其秩为$rank(A)=r, r\leq min(m,n)$，则称$U_r\sum_rV_r^T$为$A$的奇异值分解，即$A=U_r\sum_rV_r^T$。其中，$U_r$是$m\times r$矩阵，$V_r$是$n\times r$矩阵，$\sum_r$是$r$阶对角矩阵；矩阵$U_r$由完全奇异值分解中$U$的前$r$列、矩阵$V_r$由$V$的前$r$列、矩阵$\sum_r$由$\sum$的前$r$个对角线元素得到。紧奇异值分解的对角矩阵$\sum_r$的秩与原始矩阵$A$的秩相等。
  • 设$A$为$m\times n$实矩阵，其秩为$rank(A)=r$，且$0<k<r$，则称$U_k\sum_k V_k^T$为矩阵$A$的截断奇异值分解$A\approx U_k\sum_kV_k^T$，其中$U_k$是$m\times k$矩阵，$V_k$是$n\times k$矩阵，$\sum_k$是k阶对角矩阵；矩阵$U_k$由完全奇异值分解中$U$的前$k$列，矩阵$V_k$由$V$的前$k$列，矩阵$\sum_k$由$\sum$的前$k$个对角线元素得到，对角矩阵$\sum_k$的秩比原始矩阵$A$的秩低。

15.1.3 几何解释

矩阵的奇异值分解可以看做是将其对应的线性变换分解为旋转变换、缩放变换及旋转变换的组合。

15.1.4 主要性质

$$
(1)设矩阵A的奇异值分解为A=U\sum V^T,则以下关系成立\
A^TA=(U\sum V^T)^T(U\sum V)^T=V({\sum}^T\sum)V^T \
AA^T=(U\sum V^T)(U\sum V^T)^T=U({\sum}^T\sum)U^T \
也就是说,矩阵A^TA和AA^T的特征分解存在，且可以由矩阵A的奇异值分解的矩阵表示。 \
V的列向量是A^TA的特征向量,U的列向量是AA^T的特征向量，\sum的奇异值是A^TA和AA^T的特征值的平方根。 \
(2)在矩阵A的奇异值分解中,奇异值、左奇异向量和右奇异向量之间对应关系。由A=U\sum V^T可得\
AV=U\sum \
比较这一等式两端的第j列,得到Av_j=\sigma_ju_j,j=1,2,…,n \
这是矩阵A的右奇异向量和奇异值、左奇异向量的关系。\
类似地,由A^TU=V{\sum}^T \
得到A^Tu_j=\sigma_jv_j,j=1,2,…,n\
A^Tu_j=0,j=n+1,n+2,…,m\
这是矩阵A的左奇异向量和奇异值、右奇异向量的关系。\
(3)矩阵A的奇异值分解中,奇异值\sigma_1,\sigma_2,…,\sigma_n是唯一的,而矩阵U和V是不唯一的 \
(4)矩阵A和\sum的秩相等,等于正奇异值\sigma_i的个数r(包含重复的奇异值) \
(5)矩阵A的r个右奇异向量v_1,v_2,…,v_r构成A^T的值域R(A^T)的一组标准正交基。\
因为矩阵A^T是从R^m映射到R^n的线性变换,则A^T的值域R(A^T)和A^T的列空间是相同的,v_1,v_2,…,v_r是A^T的一组标准正交基,\
因此也是R(A^T)的一组标准正交基。 \
矩阵A的n-r个右奇异向量v_{r+1},v_{r+2},…,v_n构成A的零空间N(A)的一组标准正交基\
矩阵A的r个左奇异向量u_1,u_2,…,u_r构成值域R(A)的一组标准正交基\
矩阵A的m-r个左奇异向量u_{r+1},u_{r+2},…,r_m构成A^T的零空间N(A^T)的一组标准正交基
$$

15.2 奇异值分解的计算

1）求$A^TA$的特征值和特征向量

2）求$n$阶正交矩阵，将特征向量单位化，得到单位特征向量$v_1,v_2,…,v_n$，构成$n$阶正交矩阵$V=[v_1 \space v_2 \space …v_n]$

3）求$m\times n$对角矩阵$\sum$。计算$A$的奇异值$\sigma_i=\sqrt{\lambda_i},i =1,2,…,n$，构造$m\times n$矩形对角矩阵$\sum$，主对角线元素是奇异值，其余元素是零。

4）求$m$阶正交矩阵$U$，对$A$的前$r$个正奇异值，令$u_j=\frac{1}{\sigma_j}Au_j,j=1,2,…,r$得到$U_1=[u_1 \space u_2 \space … u_r]$。求$A^T$的令空间的一组标准正交基${u_{r+1},u_{r+2},…,u_m}$，令$U_2=[u_{r+1}\space u_{r+2} \space … u_m]$，并令$U=[U_1\space U_2]$

5）得到奇异值分解，$A=U\sum V^T$

15.3 奇异值分解与矩阵近似

15.3.1 弗罗贝尼乌斯范数

  • 设矩阵$A\in R^{m\times n},A=[a_{ij}]{m\times n}$，定义矩阵$A$的弗罗贝乌斯范数为$||A||F=(\sum{i=1}^m\sum{j=1}^n(a_ij)^2)^\frac12$

  • 设矩阵$A\in R^{m\times n}$，$A$的奇异值分解为$U\sum V^T$，其中$\sum=diag(\sigma_1,\sigma_2,…,\sigma_n)$，则$||A||_F=(\sigma_1^2+\sigma_2^2+…+\sigma_n^2)^\frac12$

15.3.2 矩阵的最优近似

  • 设矩阵$A\in R^{m\times n}$，矩阵的秩$rank(A)=r$，并设$\cal M$为$R^{m\times n}$中所有秩不超过$k$的矩阵集合，$0<k<r$，则存在一个秩为$k$的矩阵$X\in \cal M$，使得$||A-X||F=min{S\in \cal M}||A-S||_F$。称矩阵$X$为矩阵$A$在弗贝尼乌斯范数意义下的最优近似
  • 设矩阵$A\in R^{m\times n}$，矩阵的秩$rank(A)=r$，有奇异值分解$A=U\sum V^T$，并设$\cal M$为$R^{m\times n}$中所有秩不超过$k$的矩阵的集合，$0<k<r$，若秩为$k$的矩阵$X\in \cal M$，满足$||A-X||F=min{S\in \cal M}||A-S||F$，则$||A-X||F=(\sigma{k+1}^2,\sigma{k+2}^2,…,\sigma_n^2)^\frac12$，特别地，如果$A’=U\sum’V^T$，其中$\sum’=\left[ \begin{array}{1}\sum_k & 0 \ 0 &0 \end{array}\right]$，则$||A-A’||F=(\sigma{k+1}^2+\sigma_{k+2}^2+…+\sigma_n^2)^\frac12=min_{S\in \cal M}||A-S||_F$

15.3.3 矩阵的外积展开式

$A=\sum_{k=1}^nA_k=\sum_{k=1}^n\sigma_ku_kv_k^T$，其中$u_kv_k^T$为$m\times n$矩阵，是列向量$u_k$和行向量$v_k^T$的外积，$\sigma_k$为奇异值，$u_k.v_k^T,\sigma_k$通过矩阵$A$的奇异值分解得到。
