Automating the Generation of 3D Multiple Pile Layout Design Using BIM and Heuristic Search Methods

概要

  • 《Automating the Generation of 3D Multiple Pipe Layout design Using BIM and Heuristic Search Methods》

内容

摘要

多管道系统的布局设计和空间协调是建筑行业面临的主要挑战。多管道布局设计的目的是在各种约束条件下，在三维环境中，为多个单独的管道找到一个最优的从不同起点到不同终点的路径布局。目前，管道布置图的设计都是由手工进行，工作量大、容易出错、耗时长。

提出了一种基于BIM启发式搜索的多管道布局设计方法。根据管道布置设计的物理要求、设计要求、经济性和安装要求，提出了基于有向加权图的算法。管道之间以及与构建组件之间的冲突被考虑在内，并在布局优化中加以避免。在此基础上，采用模拟退火(SA)算法在大搜索空间内近似全局优化求解多个管道布局优化问题。在布局设计方面，实现了Dijkstra算法和两种启发式算法3D A*和果蝇优化算法(FOA)，并进行了比较，得到了多管道系统的布局设计。

以一个典型的具有9条管路的厂房为例，说明了所提出的多管布置设计方法。结果表明，该方法能够生成最优无碰撞的多管系统布局。与传统方法相比，该方法大大减少了多管布置设计的时间和成本。

约束目标

  • 物理约束：构件和管道之间避免碰撞，管道与管道之间避免碰撞
  • 设计约束：管道之间不应有冲突，阀门附件等常用构建应留有空间，需要留有一定的空间
  • 经济约束：管道长度应该尽可能短，转弯数量应该尽可能少，安装费用应该尽可能低（需要尽可能靠近墙壁和一些支撑障碍物），支持管道间的共享

本文目标

因此，本文旨在开发一种基于自动化建筑信息建模(BIM)的启发式方法，用于三维环境下的多管布局设计优化。

BIM：是建筑学、工程学及土木工程的新工具。建筑信息模型或建筑资讯模型一词由Autodesk所创的。它是来形容那些以三维图形为主、物件导向、建筑学有关的电脑辅助设计。

适应度函数

$L_i=abs(X_i-X_{i+1})+abs(Y_i-Y_{i+1})+abs(Z_i-Z_{i+1}),C_p=C_m[\sum_{i=1}^nL_i+\sum_{j=1}^xNB_j]$，其中，$C_m$管道材料的花费，$C_p$为开始节点到结束节点的成本，$L_i$为总距离花费，$NB_j$为弯头的等效长度，这里如果是90度就看做30.

$C_I=I_C\sum_{i=1}^nd_i$，$I_C$表示安装成本每单位

$\min TCM=C_P+C_I=C_m[\sum L_i+\sum NB_j]+I_C\sum d_i$，这函数越小越优秀

问题

针对什么样的场景？

其实是比较通用的场景，论文举例是以一个小厂房为特定的例子说明。

使用了哪些算法和方法？

Dijsktra算法作为基础准则，整体框架为模拟退火和果蝇的启发式搜索。在模拟退火的大环境下，不断重复的对管线进行启发式搜索，以此来测试是否能到达最优的状态。有点特色的地方就是引入了铺设管线的成本作为启发式函数。

为什么使用这些方法？

这个问题不是np完全的，在大范围随机空间采用启发式搜索的方法来保证正确性，在外层套个模拟退火是为了进一步优化。

有哪些可以改进的地方？

启发式函数设计的很简陋，没有加入障碍物，弯折脚等考虑因素。

适合我项目的地方？

这篇论文很垃圾，方法都是简单基础的介绍，然后直接例子说明结果了，具体方法都是一笔带过。
